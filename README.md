\# 📱 WinMobiles Website



A responsive business website developed for \*\*WinMobiles\*\* that allows customers to request mobile phone purchases online.



The website collects user information and sends an automated email notification to the company using EmailJS, enabling them to contact the customer with pricing and delivery confirmation.



---



\## 🚀 Features



\- Mobile purchase request form

\- Email notification using EmailJS

\- Responsive design for mobile and desktop

\- Clean and simple user interface



---



\## 🛠️ Technologies Used



\- HTML5

\- CSS3

\- JavaScript

\- EmailJS



---



\## 🔄 How It Works



1\. User fills out the purchase request form.

2\. Form data is sent via EmailJS.

3\. Company receives customer details.

4\. Company contacts the customer via phone regarding pricing and delivery.
